# Система управления 3D-печатью

Веб-приложение для организации процесса приема и обработки заказов на 3D-печать. Система разделена на клиентскую часть для пользователей и панель управления для администраторов.

## Основные возможности

### Для пользователей
- Регистрация и авторизация в системе.
- Создание заявок на 3D-печать с указанием данных и комментариев.
- Отслеживание статуса исполнения заявок (Ожидание, В обработке, Завершено).
- Архивация старых или ненужных заявок.
- Управление профилем пользователя и удаление аккаунта.
- Быстрое переключение между несколькими аккаунтами на одном устройстве.

### Для администраторов
- Полный обзор всех поступивших заявок.
- Управление статусами заказов.
- Возможность архивации выполненных или неактуальных работ.

## Технологический стек

### Фронтенд
- React 18 с использованием TypeScript.
- Vite как инструмент сборки.
- Tailwind CSS для стилизации интерфейса.
- Motion (framer-motion) для плавной анимации.
- Lucide React для иконок.

### Бэкенд
- Express.js (Node.js сервер).
- SQLite3 в качестве базы данных.
- JWT (JSON Web Tokens) для безопасной аутентификации.
- Bcrypt для хеширования паролей.

## Автоматизация и правила
- Заявки старше 7 дней автоматически перемещаются в архив при просмотре списка заявок.
- При удалении аккаунта все связанные с ним заявки удаляются безвозвратно.
- Функция "Забыть аккаунт" позволяет удалить данные о входе из локального хранилища браузера, не удаляя сам аккаунт на сервере.

## Установка и запуск

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Запустите сервер разработки:
   ```bash
   npm run dev
   ```

3. Соберите проект для публикации:
   ```bash
   npm run build
   ```

## Структура проекта
- /src - Исходный код фронтенда (компоненты, страницы, сервисы).
- /server - Код бэкенда, маршруты и конфигурация базы данных.
- /public - Статические ресурсы проекта.
